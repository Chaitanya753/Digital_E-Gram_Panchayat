## 🚀 Digital E-Gram Panchayat System

A web-based Digital E-Gram Panchayat Management System that allows citizens to apply for government services online such as Birth Certificate, Death Certificate, Income Certificate, Residence Certificate, and Government Scheme Applications.

This system digitizes village-level administrative services, reduces paperwork, increases transparency, and improves efficiency in service delivery.

## 📦 Installation & Setup
🔹 Step 1: Clone Repository
git clone https://github.com/Chaitanya753/Digital_E-Gram_Panchayat.git
🔹 Step 2: Move to Project Folder
cd digital-egram-panchayat
🔹 Step 3: Setup Database

Open phpMyAdmin

Create a new database:

digital_egram_panchayat

Import the provided SQL file

🔹 Step 4: Configure Database Connection

Update db_connection.php:

$conn = mysqli_connect("localhost", "root", "", "digital_egram_panchayat");
🔹 Step 5: Run Project

Start Apache & MySQL in XAMPP

Open browser:

http://localhost/Digital_E-Gram_Panchayat
